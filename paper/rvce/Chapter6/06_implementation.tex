\subsection{Standard K-Means Algorithm}

The K-Means algorithm is a clustering technique used to partition data into distinct clusters based on similarity. It iteratively assigns data points to clusters and updates centroid positions until convergence is achieved.

The steps involved in the K-Means algorithm include:
\begin{enumerate}
    \item Initialize cluster centroids
    \item Assign data points to nearest centroids
    \item Update centroid positions
    \item Repeat until convergence
\end{enumerate}

\subsection{Mapper Part of K-Means Algorithm}

This subsection explains the Mapper phase of the K-Means algorithm in a distributed processing environment. The Mapper processes input data and generates intermediate key-value pairs based on cluster assignments.

The Mapper functionality may include:
\begin{itemize}
    \item Reading input data
    \item Calculating distances from centroids
    \item Assigning clusters
    \item Generating intermediate outputs
\end{itemize}

%----------------------------------------------------------

\subsection{Reducer Part of K-Means Algorithm}

The Reducer phase processes the intermediate outputs generated by the Mapper and computes updated centroid values for each cluster.

The Reducer functionality may include:
\begin{itemize}
    \item Aggregating cluster data
    \item Calculating updated centroids
    \item Generating final clustered outputs
\end{itemize}
